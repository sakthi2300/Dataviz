📊 Data Analytics Dashboard

A full-stack Data Analytics Dashboard that enables users to upload datasets, process data, and generate meaningful insights through interactive visualizations.

🚀 Overview

This project is designed to analyze business data efficiently by transforming raw CSV files into actionable insights.

📥 Upload datasets dynamically
🧹 Clean and preprocess data
📊 Perform aggregations (sum, average, count)
📈 Visualize trends and patterns
🔍 Explore insights interactively
✨ Features
CSV file upload and parsing
Data cleaning using Pandas
Aggregation for trend analysis
Interactive charts and dashboards
Region-wise and category-wise analysis
Scalable backend with REST APIs
🛠️ Tech Stack
Python (Pandas) – Data cleaning & preprocessing
MySQL – Data storage & querying
Java (Spring Boot) – Backend APIs
Angular – Dashboard UI
Highcharts – Data visualization
PapaParse – CSV parsing
REST APIs – Communication between frontend & backend
⚙️ How It Works
User uploads a CSV dataset
PapaParse parses the file on the frontend
Backend processes and stores data in MySQL
Pandas performs cleaning and transformations
Aggregated data is sent via APIs
Highcharts displays insights visually
